# GitHub Repository Setup

## Step 1: Create the Repository
1. Go to https://github.com/nishamehndidesigner
2. Click "New repository"
3. Name it: `gallery`
4. Make it **PUBLIC**
5. Initialize with README
6. Click "Create repository"

## Step 2: Upload Images
Upload images with these exact names:

### Bridal Category
- `bridal_1.jpg`
- `bridal_2.jpg` 
- `bridal_3.jpg`
- etc.

### Party Category
- `party_1.jpg`
- `party_2.jpg`
- `party_3.jpg`
- etc.

### Arabic Category

- `arabic_2.jpg`
- `arabic_3.jpg`
- etc.

### Simple Category

- `simple_2.jpg`
- `simple_3.jpg`
- etc.

## Step 3: Verify Setup
After uploading, these URLs should work:
- Repository: https://github.com/nishamehndidesigner/gallery
- API: https://api.github.com/repos/nishamehndidesigner/gallery/contents
- Raw file: https://github.com/nishamehndidesigner/gallery/raw/main/bridal_1.jpg

## Important Notes
- Repository MUST be public
- Images MUST follow the naming convention
- Supported formats: .jpg, .jpeg, .png, .gif, .webp
